# 🐍 Doodle Snake

A hand-drawn `.io` snake game — slither around a notebook page, munch colorful doodle orbs to grow, and make rival snakes crash into you so you can eat what's left. Built from scratch as a single HTML5 Canvas file.

**▶ Play: https://olyhelpschoreo.github.io/doodle-snake**

## How to play
- **Move** — mouse or drag
- **Boost** — hold click / space (leaves a scribbly speed-trail; costs length)
- **Eat** the colorful orbs to grow longer
- Make other snakes **crash into your body** — then eat the bits they drop
- Don't hit anyone's body or the torn-paper wall

## Features
- Fully hand-drawn doodle style: wavy graph-paper world, sketchy ink outlines, crayon colors, torn-paper edges
- 16 AI bot snakes that hunt food, dodge bodies, and try to cut you off
- Live leaderboard, minimap, googly eyes + flicking tongues, boost speed-trails
- One file, no build step, no dependencies — runs in any modern browser

## Run locally
```
node server.js      # then open http://localhost:8400
```
